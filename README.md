# MALL - The Future of Retail Experience

A high-performance, cinematic web experience built to showcase a premier luxury destination. Designed with a video-first approach and advanced motion physics to create an immersive digital journey.

## 🌐 Live Experience
**Deploy Link:** [https://mall-pearl-sigma.vercel.app/](https://mall-pearl-sigma.vercel.app/)

## 🚀 Tech Stack
- **Framework:** React 19 (Vite)
- **Styling:** Tailwind CSS v4 (Modern configuration)
- **Animation:** GSAP (ScrollTrigger) & Framer Motion
- **Icons:** Lucide React & React Icons (FA)
- **Assets:** High-bitrate cinematic video & AI-curated imagery

## ✨ Key Features
- **Cinematic Loader:** A custom-built preloader that initializes the immersive environment.
- **GSAP Scroll Narrative:** Complex scroll-triggered sequences that transform the website into a digital story.
- **Video-First UI:** Dynamic video backgrounds with cinematic color-grading overlays.
- **Interactive Booking Flow:** A seamless, premium event booking experience integrated across all sections.
- **Luxury Aesthetic:** Minimalist design inspired by global brands like Apple, Hermès, and Tesla.

## 🤖 AI Usage Disclosure
In alignment with modern industry standards, this project utilized Generative AI for:
- **Visual Ideation:** Conceptualizing the layout and flow of luxury storytelling.
- **Asset Selection:** Curating cinematic video assets and high-end renderings.
- **Copywriting:** Crafting the narrative of "The Future of Retail".

## 🛠️ Setup & Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## 📈 Performance Optimization
- **Lazy Loading:** All heavy sections are deferred until needed.
- **Motion Optimization:** GSAP Context is used for efficient memory cleanup and scroll handling.
- **Responsive Layout:** Flawless experience across Desktop, Tablet, and Mobile.
